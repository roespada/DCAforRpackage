# DCAforRpackage
R package for mfDCA and mfDCAid calculations


# Installation

You must have R installed.
If you do not have devtools installed, you can get it from CRAN.
In R type:

     library(devtools)

     install_github("roespada/DCAforRpackage",subdir="DCAforR")

     library(DCAforR)
